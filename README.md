# Dağıtık Sosyal Medya Analiz Platformu

## Proje Açıklaması

Bu proje, sosyal medya platformlarından veri toplayan ve bu verileri gerçek zamanlı olarak analiz eden dağıtık bir sistem geliştirmeyi amaçlamaktadır. Sistem, sosyal medya paylaşımlarındaki trend konuları belirler ve paylaşımlar üzerinde duygu analizi gerçekleştirir.

## Kullanılan Teknolojiler

* Java
* Spring Boot
* Apache Kafka
* Apache Spark
* Elasticsearch
* AWS / Azure / GCP

## Proje Yapısı

* **docs/** → Proje dokümantasyonu
* **diagrams/** → Sistem mimarisi diyagramları
* **backend/** → Backend servisleri
* **frontend/** → Web arayüzü

## Özellikler

* Sosyal medya verilerinin toplanması
* Gerçek zamanlı veri işleme
* Duygu analizi
* Trend konuların belirlenmesi
* Veri görselleştirme paneli
